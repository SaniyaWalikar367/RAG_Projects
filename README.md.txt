# RAG Project (Retrieval Augmented Generation)

## Description
This project demonstrates a basic RAG (Retrieval Augmented Generation) pipeline using Python and LangChain.

## Files
- sandy.py : Main application file
- Basic.py : Basic Python logic
- RAG_langchain.py : LangChain based RAG implementation

## Requirements
Install dependencies using:
pip install -r requirements.txt

## How to Run
python sandy.py
